# Home Lab SOC Project

A cybersecurity home lab simulating real-world attacks and
detection using Kali Linux, Metasploitable 2, and Splunk.

## Lab Environment

- Attacker: Kali Linux
- Target: Metasploitable 2
- Network: VirtualBox Host-Only (isolated)
- SIEM: Splunk Free

## Project Phases

1. Environment Setup
2. Reconnaissance
3. Exploitation
4. Detection & Alerting in Splunk
5. Final Report
